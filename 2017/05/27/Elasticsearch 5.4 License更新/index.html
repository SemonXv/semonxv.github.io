Elasticsearch 5.4ES5 License更新

申请免费 license

step 01 官网填写申请信息

https://license.elastic.co/registration/

step 02 邮箱查看license 下载地址

step 03 elasticsearch 安装 x-pack

bin/elasticsearch-plugin install x-pack

x-pack 初始账号

user:   elastic
pwd:    changeme

step 03 更新 licens

curl -XPUT -u elastic 'http://192.168.1.112:9200/_xpack/license' -H "Content-Type: application/json" -d @license.json

注意：中间要求输入 elastic 的密码，license.json为license.json文件路径
